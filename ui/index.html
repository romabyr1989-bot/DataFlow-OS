Конвейер — это цепочка шагов, которая выполняется автоматически по расписанию. Каждый шаг может загрузить данные (из CSV, HTTP, БД) и/или трансформировать их через SQL, сохранив результат в таблицу. Нажмите ▶ Запустить для ручного старта или настройте cron-расписание для автозапуска.
